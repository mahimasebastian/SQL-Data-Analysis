# SQL Data Analysis using Olist E-commerce Dataset

## Project Overview

This project demonstrates SQL data analysis using the Olist Brazilian E-commerce dataset. The objective is to practice querying and analyzing structured data with SQL by performing filtering, sorting, grouping, joins, aggregate functions, subqueries, views, and query optimization.

## Objectives

* Learn SQL data manipulation and analysis.
* Extract meaningful insights from an e-commerce database.
* Practice writing efficient SQL queries.
* Understand database relationships using joins.
* Perform business analysis using aggregate functions.

## Tools Used

* MySQL Workbench
* SQL
* Olist Brazilian E-commerce Dataset
* GitHub

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* INNER JOIN
* LEFT JOIN
* Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
* Subqueries
* Views
* Indexes

## Dataset

The project uses the Olist Brazilian E-commerce Public Dataset, which contains customer, order, product, payment, seller, and review information for performing SQL-based data analysis.

## Project Structure

* `dataset/` – CSV dataset files
* `queries.sql` – SQL queries used for analysis
* `screenshots/` – Query output screenshots
* `SQL_Report.docx` – Project report
* `README.md` – Project documentation

## Outcome

This project demonstrates the ability to query, analyze, and extract business insights from a relational database using SQL. It serves as a beginner-friendly SQL portfolio project suitable for internship submissions and GitHub portfolios.
